# The Vende
Download Facebook album with single click and Backup it with Google Drive.

# Required Library:
-------- Designing Purpose--------
* [Twitter Bootstrap]
* [jQuery: JavaScript Library]
* [Font Awesome]

-------- Development Purpose--------
* [API Client Libraries (PHP)]
* [Facebook Graph SDK v5 for PHP]
